# InfoHub Challenge

A multi-functional web application that includes:
- Weather Module
- Currency Converter
- Quote Generator

## Project Structure
```
InfoHub-Challenge/
├─ client/           # React frontend
├─ api/              # Backend API routes
├─ tailwind.config.js
├─ postcss.config.js
└─ README.md
```

## Setup and Installation

1. Install dependencies:
```bash
# Install client dependencies
cd client
npm install

# Install API dependencies
cd ../api
npm install
```

2. Run the application:
```bash
# Start the frontend (from client directory)
npm run dev

# Start the backend (from api directory)
npm start
```

## Features
- Real-time weather information
- Currency conversion
- Random quote generation
